# 🎯 나만의 퀴즈 게임

> Python 프로그래밍 상식 퀴즈를 풀고, 추가하고, 점수를 기록하는 터미널 게임입니다.

---

## 📌 프로젝트 개요

Python 기본 문법(변수, 조건문, 반복문, 함수, 클래스, 파일 입출력)을 직접 활용하여
터미널에서 동작하는 퀴즈 게임을 처음부터 끝까지 구현한 프로젝트입니다.

---

## 🧠 퀴즈 주제 및 선정 이유

**주제: Python 프로그래밍 상식**

이 과제 자체가 Python을 배우는 과정이기 때문에,
퀴즈를 풀면서 배운 내용을 복습할 수 있도록 Python 관련 문제를 선정했습니다.
학습과 실습이 동시에 이루어지는 시너지 효과를 기대했습니다.

---

## ▶️ 실행 방법

```bash
# 1-2 디렉터리로 이동
cd 1-2

# 게임 실행 (Python 3.10 이상 필요)
python main.py
```

---

## 🗂️ 기능 목록

| 번호 | 기능 | 설명 |
|------|------|------|
| 1 | 퀴즈 풀기 | 저장된 퀴즈를 랜덤 순서로 출제, 정답/오답 즉시 피드백 |
| 2 | 퀴즈 추가 | 문제, 선택지 4개, 정답 번호를 입력해 새 퀴즈 등록 |
| 3 | 퀴즈 목록 | 등록된 모든 퀴즈의 문제를 번호와 함께 나열 |
| 4 | 점수 확인 | 지금까지의 최고 점수(정답 수 / 전체 문제 수) 확인 |
| 5 | 종료 | 프로그램 안전 종료 (데이터 자동 저장) |

## 🗄️ 데이터 파일 설명 (state.json)

- **경로**: 프로젝트 루트(`1-2/state.json`)
- **인코딩**: UTF-8
- **역할**: 퀴즈 목록과 최고 점수를 영속적으로 저장
- **스키마**:

```json
{
  "quizzes": [
    {
      "question": "문제 내용",
      "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
      "answer": 1
    }
  ],
  "best_score": {
    "correct": 4,
    "total": 5
  }
}
```

| 필드 | 타입 | 설명 |
|------|------|------|
| `quizzes` | list | 퀴즈 객체 배열 |
| `quizzes[].question` | str | 문제 문자열 |
| `quizzes[].choices` | list[str] | 4개의 선택지 |
| `quizzes[].answer` | int | 정답 번호 (1~4) |
| `best_score.correct` | int | 최고 기록 정답 수 |
| `best_score.total` | int | 해당 게임의 전체 문제 수 |
